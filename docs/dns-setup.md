# DNS Setup

Domain registered via Namecheap: threatlensuae.xyz

## A Records configured

| Host | Type | Value | TTL |
|------|------|-------|-----|
| @ | A | 135.235.217.244 | Automatic |
| www | A | 135.235.217.244 | Automatic |

## Steps taken
1. Purchased domain threatlensuae.xyz via Namecheap
2. Navigated to Domain List > Manage > Advanced DNS
3. Added an A Record with Host "@" pointing to 135.235.217.244
4. Added a second A Record with Host "www" pointing to 135.235.217.244
5. Verified propagation using dnschecker.org
